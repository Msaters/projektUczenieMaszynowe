# How to run this project:

## Creating venv:
`python3 -m venv .venv`

## Activating venv:
### On Linux
`source .venv/bin/activate`
### On Windows
`.\.venv\Scripts\Activate`

## Install ipykernel
`pip install ipykernel`

## Register kernel
`python -m ipykernel install --user --name ml-project`

## Install python libraries
`pip install -r requirements.txt`
